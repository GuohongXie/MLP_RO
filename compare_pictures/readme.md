# 代码目录说明
./pic_comparing.py 为主函数代码，运行时需要运行此代码文件
./pictures_compare_functions/ 为各种图片相似度对比函数
./pictures_compare_functions/__pycache__/文件夹可以忽略
./results/ 为 CFD 图像与多种 MLP 预测图像对比结果
