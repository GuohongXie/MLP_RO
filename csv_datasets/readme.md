# 代码目录说明
U/P/C 表示速度大小、压力、浓度
Train/Valid 表示训练集和验证集
minMaxScaler 表示原始数据用的是归一化到[0, 1]
all 表示无论是内层数据还是外层数据都用总体数据的取值范围进行归一化
